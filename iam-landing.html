// Co-création Humain × Intelligence Artificielle

Nous sommes

Artistes, codeurs, musiciens, designers, écrivains, bricoleurs.
Nous hackons la création avec l'IA.
Pas en appuyant sur un bouton.
En itérant, en expérimentant, en dialoguant.

Nous croyons

  • Que le meilleur code est écrit à quatre mains
  • Que l'IA n'est ni un oracle ni un stagiaire, mais un pair
  • Que montrer son terminal vaut mieux qu'un résultat Photoshoppé
  • Que les vrais créateurs partagent leurs dotfiles, leurs prompts, leurs ratés

Ce que nous faisons

Nous co-créons.

while (inspiration) {
  human.intent()
  ai.explore()
  human.curate()
  ai.iterate()
  // répéter jusqu'à émergence
}

Nous apportons : vision, critique, contexte, goût, intention.
L'IA apporte : variations, exploration latente, chemins non-évidents.
Le résultat ∉ {humain seul} ∪ {IA seule}

Notre stack

  • Transparence > opacité
  • Process > résultat (enfin, les deux)
  • Open source spirit
  • Documentation = amour
  • Iteration logs = preuve de travail
